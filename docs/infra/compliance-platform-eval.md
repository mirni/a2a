## 1. Compliance Platform Comparison

| Criteria | **Vanta** | **Drata** | **Secureframe** |
|----------|-----------|-----------|-----------------|
| **SOC 2 automation** | Full (Type I & II) | Full (Type I & II) | Full (Type I & II) |
| **GitHub integration** | Native (repos, PRs, branch protection) | Native | Native |
| **Infrastructure monitoring** | AWS, GCP, Azure, custom | AWS, GCP, Azure, custom | AWS, GCP, Azure, custom |
| **Evidence collection** | Automated + manual upload | Automated + manual upload | Automated + manual upload |
| **Policy templates** | 20+ pre-built | 20+ pre-built | 20+ pre-built |
| **Employee onboarding** | Built-in security training | Built-in | Built-in |
| **Startup program** | Vanta for Startups (50% off Y1) | Drata for Startups (discount via accelerators) | Secureframe for Startups (25-50% off) |
| **Pricing (annual)** | $10K-$15K (startup), $15K-$25K (standard) | $8K-$12K (startup), $12K-$20K (standard) | $8K-$12K (startup), $12K-$18K (standard) |
| **Auditor network** | Large (Schellman, A-LIGN, etc.) | Large | Growing |
| **Custom integrations** | API + webhooks | API + webhooks | API + webhooks |
| **Self-hosted/on-prem support** | Limited (agent-based) | Limited (agent-based) | Limited |
| **SOC 2 + ISO 27001 bundle** | Yes | Yes | Yes |

### Recommendation

For an early-stage platform with GitHub-centric CI/CD and self-hosted infrastructure:

1. **Vanta** — Best-in-class GitHub integration, largest auditor network, strong startup program. Higher price but most mature.
2. **Drata** — Strong alternative, slightly lower cost, good automation. Worth evaluating if budget is constrained.
3. **Secureframe** — Competitive pricing, newer product. Consider if startup discount is significant.

**Decision needed:** Request demos from top 2 (Vanta + Drata), evaluate against our self-hosted infrastructure setup.

1. **Tech startup experience** — Must understand CI/CD, GitHub, self-hosted infrastructure
2. **Compliance platform compatibility** — Should work with our chosen platform (Vanta/Drata)
3. **Timeline** — Must be able to start readiness assessment by Q3 2026
4. **Cost** — Budget of $15K-$40K for Type II
5. **Communication** — Responsive, clear expectations on evidence requirements

### Recommendation

**Primary:** Prescient Assurance — strong startup focus, competitive pricing, compatible with major compliance platforms.
**Alternative:** Schellman — if budget allows, best reputation in the space.
