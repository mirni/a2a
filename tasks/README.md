# Instructions on how to use the backlog
* Do not process files in here automatically, wait for human prompt to process one or several files from here.
* When the prompt is to process several files, follow the order requested in prompt, or lexicographic order of filenames otherwise.
* After finishing the backlog/ task, move the task to done/ and make a short entry in MASTER_LOG.md with the filename of the task that was just finished.
